~ How to Set It Up Yourself

Step 1: Install Claude Code

Install the Claude Code CLI from Anthropic. This is the terminal-based AI assistant that powers the skill.

# Install Claude Code
npm install -g @anthropic-ai/claude-code

Step 2: Create a GitHub repo

Create a repo for your pages. This is where your HTML files live.

# Create and init repo
mkdir my-pages && cd my-pages
git init && git remote add origin [your-repo-url]

Step 3: Connect to Cloudflare Pages

Go to Cloudflare Dashboard, create a new Pages project, and connect it to your GitHub repo. Every push to main auto-deploys. Free tier, unlimited bandwidth.

Step 4: Create the skill file

Add a skill file at .claude/skills/publish-page/SKILL.md in your project. This tells Claude your brand colors, fonts, templates, repo path, and deploy process. Claude reads this file whenever you invoke the skill.

Step 5: Say the magic words

# That's it. Just say:
"Publish page for [whatever you need]"
